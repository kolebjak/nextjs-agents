# Nextjs with Mastra 

- NextJS clean installation
- Install Mastra `yarn add -D mastra`
- Added resolutions in package.json for post-hog, default one wasn't working
```
 "resolutions": {
    "posthog-node": "4.14.0"
  },
```

- Init Mastra in project `yarn mastra init`
- Run Mastra server `mastra dev`
- Run nextjs server
